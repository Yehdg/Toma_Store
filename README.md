# 電商網站 (Toma Store)

## 🌟 專案簡介
這是一個全端 Web 應用程式，提供番茄商品的線上購物平台。

## 🛠️ 技術棧
**前端：**
- Vue.js 2
- Vue Router
- CSS3
- Axios


**後端：**
- Node.js
- Express.js
- Prisma ORM
- mySql2
- Prisma


**資料庫：**
- My SQL


## 🚀 功能特色
- 👤 會員註冊/登入
- 🍅 商品瀏覽
- 🛒 購物車管理
- 📸 頭像上傳
- 🔒 密碼修改
- 📋 訂單歷史

# 安裝依賴
cd backend
npm install
npm start

cd frontend  
npm install
npm run serve

## 🎯 專案特色說明
- **響應式設計**：支援各種裝置瀏覽
- **檔案上傳**：支援頭像上傳功能
- **購物車管理**：完整的購物流程
